# scientific-computing
Scientific Computing &amp; Numerical Methods in Fortran

Scientific Computing in Fortran

This repository contains implementations of classical numerical methods
commonly used in scientific computing and computational physics.

The goal of the project is to explore how mathematical models can be
implemented computationally using the Fortran language.

Implemented topics include:

• root finding algorithms
• numerical differentiation
• numerical integration
• interpolation methods
• differential equation solvers
• statistical analysis
